# ☕ Estudos Java - Programação Orientada a Objetos (POO)

Repositório dedicado aos meus estudos de **Java com foco em Programação Orientada a Objetos (POO)**.

A proposta deste repositório é **registrar minha evolução prática**, começando pelos conceitos mais básicos e avançando gradualmente, exercício por exercício.

---

## 🚀 Objetivo

* Consolidar a base de Java
* Aprender e praticar POO na prática
* Evoluir de exercícios simples para estruturas mais próximas do mundo real
* Construir uma base sólida para projetos maiores no futuro

---

## 🧠 Conceitos abordados

Ao longo dos exercícios, venho praticando:

* Classes e Objetos
* Atributos e Métodos
* Construtores
* Parâmetros
* Encapsulamento (`private`, getters)
* Validação de dados
* Interação entre objetos
* Organização de código

---

## 📈 Evolução

A ideia **não é ter código perfeito desde o início**, mas sim mostrar:

* Como comecei
* Como fui melhorando
* Como os conceitos foram sendo aplicados

Esse repositório funciona como um **diário de aprendizado prático**.

---

## 🛠️ Tecnologias

* Java
* IntelliJ IDEA
* Git & GitHub

## 📌 Próximos passos
