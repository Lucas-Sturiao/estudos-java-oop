# Exercício 10: Contagem Regressiva com Laço For

### 📝 Proposta
Crie um programa utilizando os conceitos de estruturas de repetição em Java que atenda ao seguinte requisito:

* **Comportamento:** Utilize o laço `for` para realizar uma contagem regressiva.
* **Saída:** O programa deve imprimir no console os números de **10 até 1**, em ordem decrescente.

### 🚀 Execução (App)
1. No método principal, estruture a sintaxe do laço `for` configurando a variável inicial em 10.
2. Defina a condição de parada para que o laço continue enquanto a variável for maior ou igual a 1.
3. Utilize o operador de decremento (`--`) para reduzir o valor a cada iteração.

---
*Foco: Praticar a lógica de decremento em laços de repetição e controle inverso de fluxo.*
