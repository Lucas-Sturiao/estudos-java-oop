# Exercício 05: Classe Livro e Construtores

### 📝 Proposta
Crie uma classe chamada **Livro** com os seguintes requisitos:

**Atributos:**
* `titulo`
* `autor`
* `paginas`

**Construtor:**
* Deve receber `titulo`, `autor` e `paginas` como parâmetros.
* Deve atribuir os valores recebidos aos atributos da classe utilizando a palavra-chave **`this`**.

**Método:**
* `mostrarInformacoes()`: Imprime o Título, Autor e a quantidade de Páginas do livro.

### 🚀 Execução:
1. Instancie um objeto do tipo **Livro** passando os dados diretamente pelo **construtor**.
2. Chame o método `mostrarInformacoes()` para exibir os detalhes do livro criado.

---
*Foco: Praticar a inicialização de objetos via construtores e o uso do ponteiro `this` para diferenciação de escopo.*
