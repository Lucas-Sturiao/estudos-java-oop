# Exercício 09: Contagem Simples com Laço For

### 📝 Proposta
Crie um programa utilizando os conceitos de estruturas de repetição em Java que atenda ao seguinte requisito:

* **Comportamento:** Utilize o laço `for` para realizar uma contagem progressiva.
* **Saída:** O programa deve imprimir no console os números de **1 até 10**, um abaixo do outro.

### 🚀 Execução (App)
1. No método principal, estruture a sintaxe do laço `for` definindo a inicialização, a condição de parada e o incremento.
2. Execute o código para validar a exibição da sequência numérica completa no console.

---
*Foco: Praticar a sintaxe do laço de repetição `for` e controle de iterações.*
