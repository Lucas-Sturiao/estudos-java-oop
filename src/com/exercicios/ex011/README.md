# Exercício 11: Números Pares com Laço For

### 📝 Proposta
Crie um programa utilizando os conceitos de estruturas de repetição e condicionais em Java que atenda ao seguinte requisito:

* **Comportamento:** Utilize o laço `for` para percorrer uma sequência numérica.
* **Saída:** O programa deve filtrar e imprimir no console apenas os números **pares** no intervalo de **0 até 20** (inclusive).

### 🚀 Execução (App)
1. No método principal, estruture o laço `for` para iniciar em 0 e ir até 20.
2. Dentro do laço, utilize uma estrutura condicional (`if`) com o operador de resto da divisão (`%`) para verificar se o número é par.
3. Imprima o número caso ele atenda à condição.

---
*Foco: Praticar a combinação de laços de repetição com estruturas condicionais e operadores aritméticos.*
