# Exercício 04: Classe Aluno e Cálculo de Média

### 📝 Proposta
Crie uma classe chamada **Aluno** com os seguintes requisitos:

**Atributos:**
* `nome`
* `nota1`
* `nota2`

**Métodos:**
* `adicionarNota1()`: Atribui o valor à primeira nota.
* `adicionarNota2()`: Atribui o valor à segunda nota.
* `calcularMedia()`: Calcula a média aritmética das notas.
* `mostrarMedia()`: Imprime o nome do aluno e o resultado.

### 🚀 Execução:
1. Instancie um objeto do tipo **Aluno**.
2. Defina o nome do aluno.
3. Adicione a nota **7.5** através do método correspondente.
4. Adicione a nota **8.0** através do método correspondente.
5. Chame o método `mostrarMedia()` para exibir o resultado final.

---
*Foco: Praticar atribuição de valores via métodos com parâmetros e lógica matemática simples.*
