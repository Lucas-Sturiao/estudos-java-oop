# Exercício 12: Soma de Intervalo Numérico com Laço For

### 📝 Proposta
Crie um programa em Java que utilize estruturas de repetição para realizar um cálculo de acumulação numérica:

* **Comportamento:** Utilize o laço `for` para percorrer os números no intervalo de **1 até 100** (inclusive).
* **Lógica:** Use uma variável acumuladora para somar o valor de cada iteração ao total acumulado.
* **Saída:** O programa deve imprimir no console apenas o **resultado final** da soma de todos os números após o término do laço.

### 🚀 Execução (App)
1. No método principal, declare uma variável inteira (ex: `int soma = 0;`) antes do laço para armazenar o total.
2. Estruture o laço `for` para iniciar em 1 e terminar em 100, incrementando de 1 em 1.
3. Dentro do laço, some o valor atual do contador à sua variável acumuladora.
4. Fora e após o encerramento do laço, imprima a mensagem com o valor total.

---
*Foco: Praticar o conceito de variáveis acumuladoras (somatórios) dentro de estruturas de repetição.*
