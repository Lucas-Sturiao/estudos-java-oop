# Exercício 02: Classe Conta Bancária

### 📝 Proposta
Crie uma classe chamada **ContaBancaria** com os seguintes requisitos:

**Atributos:**
* `titular`
* `saldo`

**Métodos:**
* `depositar()`: Aumenta o saldo pelo valor informado.
* `sacar()`: Retira o valor do saldo.
* `mostrarSaldo()`: Imprime o saldo atual.

### 🚀 Execução (App):
1. Crie uma conta.
2. Defina o titular.
3. Deposite 100.
4. Saque 30.
5. Tente sacar 100.
6. Mostre o saldo final.

---
*Foco: Praticar manipulação de atributos, métodos com parâmetros e lógica de validação.*
